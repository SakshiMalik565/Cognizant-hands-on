# JPA, Hibernate, and Spring Data JPA Comparison (Exercise 2)

## Objective
Establish a beginner-friendly theoretical comparison detailing the differences, architectural advantages, and practical applications of JPA, Hibernate, and Spring Data JPA.

## Files Included
* **Comparison.md**: Comprehensive comparison guide containing architectural explanations, advantages, a detailed comparison grid, and usage criteria.

## Topics Covered
1. Definition of JPA, Hibernate, and Spring Data JPA.
2. The core relationships between the three concepts.
3. Specific advantages of using each technology.
4. Detailed Attribute Comparison Grid (Boilerplate, Ease of Use, Developed By, typical usage).
5. Vendor criteria guidelines (when to use each).
